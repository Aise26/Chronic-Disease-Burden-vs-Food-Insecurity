# Food insecurity & Chronic Disease Burden in Massachussetts, USA (2019–2022)
Analysis of county-level food insecurity and chronic disease prevalence using CDC PLACES and USDA datasets.

# Summary
This project analyzes county-level food insecurity and chronic disease prevalence in Massachusetts using publicly available datasets from CDC PLACES, Feeding America, and the USDA. The goal is to explore associations between food insecurity, social determinants of health, and chronic disease outcomes.

# Data Sources
- CDC PLACES (2019–2022)
- Feeding America Food Insecurity Data
- USDA Food Environment Atlas

# Methods
- Data cleaning and merging across three national datasets
- Exploratory data analysis (EDA)
- Logistic regression models
- Effect modification analysis
- Random forest classification (optional)
- Visualization of county-level patterns

# Key Questions
- How does food insecurity vary across Massachusetts counties?
- What chronic diseases show the strongest associations with food insecurity?
- Do social determinants of health modify these relationships?

# Skills Demonstrated
- Data cleaning and wrangling
- Regression modeling
- Machine learning (optional)
- Public health interpretation
- Reproducible research workflow
- Visualization and storytelling

# How to Run the Code
All code is written in R and can be run in RStudio.  
Datasets are publicly available and linked in the `data/raw/` folder.
